## Getting Started

### Prerequisites

Make sure npm is up to date:  
  ```sh
  npm install npm@latest -g
  ```

### Installation

Clone the repository:
   ```sh
   git clone https://github.com/pledoit/ledoit.dev.git
   ```
Install the required NPM packages:
   ```sh
   npm install
   ```
